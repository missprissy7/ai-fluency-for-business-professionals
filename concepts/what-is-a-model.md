# What is a Model?

**The short answer:** An AI model is the underlying system that powers an AI tool. Different models have different strengths, and knowing roughly which to use for which task makes you more effective.

## The Analogy

Think of AI tools (ChatGPT, Claude, Perplexity) as the interface - like different apps on your phone. The model is what runs underneath - like the operating system. The same app can run on different systems and behave differently.

## Models You Will Encounter

| Model Family | Made By | Accessible Through |
|-------------|---------|-------------------|
| GPT (e.g., GPT-4o) | OpenAI | ChatGPT, many apps |
| Claude (e.g., Claude 3) | Anthropic | Claude.ai |
| Gemini | Google | Google products |
| Llama | Meta | Various apps |

## Do You Need to Know Which Model to Use?

Not at first. When you are starting out, pick one tool (we recommend starting with Perplexity or Claude) and get comfortable with it. As you develop skills, you will naturally start noticing differences and develop preferences for different task types.

## What Actually Matters More Than the Model

Your prompt quality matters far more than which model you use. A well-crafted prompt on a mid-range model will almost always outperform a vague prompt on the most advanced model available. Master prompting first.
