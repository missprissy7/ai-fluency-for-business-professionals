# What is Markdown?

**The short answer:** Markdown is a simple way to format text using plain characters. It is what makes text bold, creates headers, and builds tables in GitHub and many other tools.

## Why You Need to Know This

All the files in this curriculum are written in Markdown (.md files). When you view them on GitHub, they render with formatting. When you edit them in a text editor, you see the plain characters.

## Basic Markdown Syntax

**Headers:**
# Big Header (H1)
## Medium Header (H2)
### Smaller Header (H3)

**Bold and Italic:**
**bold text**
*italic text*

**Lists:**
- Bullet point
- Another bullet

1. Numbered item
2. Another numbered item

**Checkboxes:**
- [ ] Unchecked box
- [x] Checked box

**Links:**
[Link text](https://example.com)

**Tables:**
| Column 1 | Column 2 |
|----------|----------|
| Cell 1   | Cell 2   |

## You Do Not Need to Master This

You will mostly be reading Markdown files, not writing them. The key thing is knowing that the symbols you see (like # or **) are formatting instructions, not typos. When in doubt, view the file on GitHub where everything renders cleanly.
