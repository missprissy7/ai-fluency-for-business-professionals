# 🤖 AI Fluency for Business Professionals

> **Learn to use AI properly — through doing, not watching.**

Welcome to the AI Fluency for Business Professionals curriculum. This is not a "What Is AI?" course. This is hands-on training that teaches you to use AI as a real work tool through guided tasks, reflection, and real deliverables.

---

## 🎯 What This Is

A 4-week, task-based learning program delivered entirely through GitHub Issues. Each issue is an assignment. You complete the task, post your deliverable in a comment, and close the issue. That's how you learn.

**This is for you if:**
- You're a business professional (project manager, consultant, operations leader, or similar)
- You've heard about AI but haven't built real skills yet
- You want to use AI to improve your actual work — not just play with it
- You learn by doing, not by watching videos or reading slides

**This is NOT for you if:**
- You're looking for a history or theory of AI
- You want a certification without putting in real work
- You're already an advanced AI power user
