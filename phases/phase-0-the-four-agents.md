# Phase 0: The Four Agents (Orientation Week)

> **Goal:** Understand the four AI tools you will use throughout this curriculum — what they are, what they are good at, and what they have in common. This phase runs before Week 1 and takes less than two hours.

## Overview

Before you start building AI skills, you need a map. There are hundreds of AI tools, but four power most professional AI work right now: **ChatGPT, Claude, Gemini, and Perplexity.** This curriculum is built to work with all four.

This is not a history lesson. By the end of this phase, you will have accounts set up, you will have run your first comparison task, and you will understand the one truth that makes everything else easier:

**The core skills transfer. The tools change.**

What you learn about prompting, iterating, verifying, and directing AI works on every one of these tools. Learning one well makes you faster on all of them.

| Agent | Made By | Best Known For | Free Tier |
|-------|---------|---------------|-----------|
| **ChatGPT** | OpenAI | Broadest general use; largest ecosystem of add-ons | Yes |
| **Claude** | Anthropic | Accuracy, long documents, lowest hallucination rate | Yes |

---

## What They All Have in Common

This is the most important thing in Phase 0. All four agents share the same core behaviors:

1. **They respond to how you ask.** Specific, structured prompts produce better results than vague ones — on every tool.
2. **They can be steered mid-conversation.** You can redirect, correct, and refine within the same conversation — on every tool.
3. **They can be wrong.** Confidently delivered incorrect information (hallucination) happens on all four. Verification is always required.
4. **They improve through iteration.** The first response is a draft. The fifth iteration is usually far better — on every tool.
5. **They have no memory by default.** Each new conversation starts fresh unless you provide context — on every tool.

These five behaviors are what this entire curriculum is built around. Every phase, every activity, every deliverable traces back to one of them.

## Setup Activities

### Setup 0.1: Create Your Four Accounts (30 minutes)

**Objective:** Get all four tools ready before the curriculum begins.

**Steps:**

1. Go to each URL and create a free account:

   | Tool | URL |
   |------|-----|
   | ChatGPT | https://chatgpt.com |
   | Claude | https://claude.ai |
   | Gemini | https://gemini.google.com |
   | Perplexity | https://perplexity.ai |

2. For each tool, after you sign up:
   - Start a new conversation
   - Type: "Hello. I am just getting started with AI tools. What is one thing most new users get wrong?"
   - Read the response

3. Do not evaluate the responses yet. Just get comfortable that each tool opened and responded.

**Note:** If you already have accounts on some of these, skip that tool's sign-up and move directly to the test message.

**Deliverable:** Post in the issue comment: Confirm which four accounts you now have access to.

---

### Setup 0.2: Your First Cross-Agent Comparison (45 minutes)

**Objective:** Run the same task on all four tools and observe how they differ.

This is the activity that makes everything click. Same question. Four different responses. You will immediately understand why knowing multiple tools matters.

**Steps:**

1. Choose a topic you know reasonably well from your own work. Something specific enough to evaluate — not "business" but "writing a client follow-up email after a proposal meeting."

2. Ask each of the four tools the exact same question. Copy and paste so the wording is identical:

   *"You are helping a business professional. Write a follow-up email to send to a potential client 48 hours after presenting a proposal. The tone should be confident but not pushy. Keep it under 150 words."*

3. Read all four responses carefully.

4. Fill out this comparison:

   | | ChatGPT | Claude | Gemini | Perplexity |
   |--|---------|--------|--------|-----------|
   | Tone — did it match? | | | | |
   | Length — under 150 words? | | | | |
   | Would I actually send this? | | | | |
   | What stood out? | | | | |

5. Pick the response you liked best. Ask that tool's version: "Make it even shorter — 80 words maximum, same tone."

**Reflection Questions:**
- Which tool gave you the most immediately usable response?
- What was different about how they handled the tone instruction?
- Did any tool do something the others did not?

**Deliverable:** Post in the issue comment: Which tool's response would you actually send? What was one specific difference you noticed between any two tools?

---

### Setup 0.3: Know Your Home Base (15 minutes)

**Objective:** Pick one tool as your primary working environment for the curriculum.

You will use all four tools throughout this program. But having a home base — one tool you default to — removes friction and speeds your learning.

**Steps:**

1. Based on your comparison in Setup 0.2, choose one tool as your primary for Phases 1–3. You can always switch later.

2. Consider these decision rules:
   - If you live in Google Workspace (Gmail, Docs, Drive): **Gemini** has native integration
   - If you need research with cited sources most often: **Perplexity**
   - If accuracy and long-document work matters: **Claude**
   - If you want the broadest general use and largest community: **ChatGPT**

3. Record your choice. You do not need to justify it. You can change it.

**Deliverable:** Post in the issue comment: Which tool is your home base and why? (One sentence is enough.)

## The Core Skill Map

Before moving to Phase 1, understand what you are building toward. The skills in this curriculum stack like this:

```
Phase 0 → You know the tools and have accounts
Phase 1 → You can have a productive AI conversation on any tool
Phase 2 → You can structure prompts that consistently work
Phase 3 → You apply AI to real work
Phase 4 → You design AI into your workflows
Phase 5 → You know which tool to use for which job and why
```

You are at the bottom of the stack right now. By the time you reach Phase 5, you will be able to walk into any conversation about AI — with a colleague, a client, or a hiring manager — and speak from experience, not theory.
