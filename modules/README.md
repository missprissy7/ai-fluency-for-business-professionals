## Design System

All modules share a unified design system:

- **Fonts:** DM Serif Display (headings) + DM Sans (body) via Google Fonts
- **Default theme:** Dark (toggle available in nav)
- **Navigation:** Arrow keys ← → | Home / End | dot nav | prev/next buttons
- **Facilitator notes:** Press `F` or click pencil icon (bottom-right)
- **Transition:** Slides animate with translateX + opacity, 380ms cubic-bezier

## The 3P Framework

The business curriculum (M1–M4) is organized around:

- **Products** — the AI tools themselves (M1–M2)
- **People** — how individuals use them in their roles (M3)
- **Process** — how organizations embed, govern, and measure AI (M4)

The **Personal Playbook** is a standalone product — separate brand, separate audience, can be sold or delivered independently.
